# PRODIGY_CS_01

## Caesar Cipher Tool

A versatile tool for encrypting and decrypting text using the classical Caesar cipher algorithm.

**Features:**
- **Command-Line and GUI Versions:** Originally built as a CLI tool, it was later enhanced with a `tkinter`-based graphical user interface for ease of use.
- **Custom Shift Value:** Allows users to specify any integer shift value for encryption and decryption.
- **Clear Input/Output:** Provides a clean interface for entering text and viewing the processed result.
